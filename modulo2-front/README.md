# Módulo 2 - Portafolio Frontend

Este módulo demuestra el uso de:

- HTML5  
- CSS  
- Bootstrap  
- JavaScript  
- jQuery  

Incluye un sitio web básico, responsivo y con eventos interactivos.

## Contenidos
- index.html  
- estilos.css  
- script.js  
- Carpeta img/  

## Cómo abrir el proyecto
Solo abre `index.html` en tu navegador.

## Evidencia
Este módulo corresponde a la implementación práctica del portafolio utilizando el stack JavaScript.
