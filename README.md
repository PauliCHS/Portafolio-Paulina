# Portafolio Web – Paulina Huentel

## Descripción
Este portafolio corresponde a un sitio básico que redirige a mis principales perfiles de GitHub y Linkedin, donde muestra mi formación académica, experiencia laboral, proyectos y habilidades técnicas en desarrollo web y análisis de datos. 
El sitio es responsive, interactivo y fácil de navegar.

## Tecnologías utilizadas
- HTML5, CSS3, Bootstrap
- JavaScript y jQuery
- Python (proyectos integrados)
- Django (proyecto incluido)
- Git y GitHub para control de versiones

## Contenido del portafolio
1. **Inicio:** Presentación personal y objetivos.
2. **Formación Académica:** Estudios y certificaciones.
3. **Experiencia Laboral:** Resumen de roles y proyectos clave.
4. **Proyectos:** Descripción de proyectos en Python y Django.
5. **Contacto:** Formulario de contacto e información de LinkedIn.
