Módulo 1 – Diseño Inicial del Portafolio Personal

Este módulo corresponde al primer acercamiento al diseño y planificación del portafolio web personal que se desarrollará a lo largo del curso. El objetivo principal fue definir el propósito del portafolio, su público objetivo y la estructura general del sitio, además de preparar un prototipo inicial que sirviera como guía para las siguientes etapas del proyecto.

Propósito del Portafolio

El portafolio tiene como finalidad mostrar mis avances, proyectos y habilidades técnicas desarrolladas durante el bootcamp. Es una herramienta pensada para presentarme profesionalmente y evidenciar mi crecimiento en áreas como desarrollo web, Python, análisis de datos, bases de datos y desarrollo de aplicaciones con frameworks como Django.

Público Objetivo

Este portafolio está dirigido principalmente a:
Reclutadores y empresas del rubro tecnológico
Profesionales con quienes pueda colaborar
Docentes y evaluadores del proceso formativo

Estructura Definida

El diseño inicial del portafolio contempla las siguientes secciones:

Página Principal: presentación breve sobre mí, mi formación y mis áreas de interés.
Proyectos: galería que reunirá los desarrollos realizados en los módulos del curso.
Habilidades: resumen de las herramientas y tecnologías que manejo y estoy incorporando.
Contacto: sección para facilitar comunicación profesional (correo o LinkedIn).
Diseño y Prototipo

Se creó un prototipo visual utilizando Figma, simulando una página web compuesta por imágenes, bloques de texto y secciones organizadas. Este prototipo sirve como referencia visual para el desarrollo del portafolio final.

El archivo PDF adjunto corresponde a la exportación del prototipo desde Figma.

Tecnologías Consideradas para el Desarrollo Futuro
HTML5 – estructura del sitio
CSS – estilo y personalización visual
Bootstrap – diseño responsivo
