# Módulo 8 - Portafolio Final

## Descripción
Este módulo contiene la versión final de mi **portafolio web personal**, desarrollado como proyecto integrador del curso.  
El sitio es completamente estático e incluye enlaces directos a los proyectos realizados en módulos anteriores, además de información sobre mi formación, experiencia y habilidades técnicas.

## Contenido del Portafolio
- Presentación personal
- Formación académica y certificaciones
- Experiencia profesional y logros
- Proyectos destacados (Python, SQL, Django)
- Contacto y enlaces externos

## Tecnologías utilizadas
- HTML5  
- CSS3  
- Bootstrap  
- JavaScript  
- Git y GitHub para control de versiones  

## Cómo visualizar el portafolio
Puedes visualizar el portafolio de dos formas:

### 1. Abrirlo localmente
- Clonar el repositorio:  
   ```bash
   git clone <URL_DEL_REPOSITORIO>

- Abrir el archivo index.html en tu navegador.

### 2. Publicarlo con GitHub Pages
Ir a Settings del repositorio.
Seleccionar Pages en el menú lateral.
En Branch, elegir:
- main
- / (root)
Guardar cambios.
GitHub generará un enlace público donde estará el portafolio disponible.

## Evidencias
Enlace a GitHub Pages.
Enlaces a los proyectos desarrollados en módulos anteriores dentro del propio portafolio.

## Estado del proyecto
El portafolio se encuentra finalizado, funcional y preparado para ser utilizado como presentación profesional. Sin embargo tiene la finalidad de ser actualizado y mejorado constantemente.
